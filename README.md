# Mtg

Crystal library for Magic The Gathering structs/libs

## Installation

Add the dependency to your `shard.yml`:

```yaml
dependencies:
  mtg:
    github: spoved/mtg.cr
```

Run `shards install`

## Usage

```crystal
require "mtg"
```
